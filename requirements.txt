Це веб-додаток для управління автосервісом, створення наряд-замовлень (чеків) та ведення бази клієнтських автомобілів. Розроблено для оптимізації роботи майстрів на СТО.

* **Живий пошук (Live Search):** Миттєвий пошук автомобілів та послуг по базі.
* **Каскадне управління даними:** Додавання, редагування та видалення авто (CRUD) з автоматичним підтягуванням пов'язаних послуг.
* **Калькулятор чеків (Кошик):** Динамічне формування наряд-замовлення з підрахунком загальної вартості без перезавантаження сторінки.
* **Адаптивний дизайн:** Інтерфейс повністю адаптований під ПК та мобільні пристрої (Bootstrap 5).
* **API (AJAX):** Асинхронна взаємодія фронтенду з базою даних через JavaScript Fetch API.

* **Backend:** Python 3, Flask, SQLAlchemy (ORM)
* **Database:** SQLite
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla), Bootstrap 5
* **Архітектура:** MVC, RESTful API підхід

1. Клонуйте репозиторій:
   `git clone https://github.com/ВАШ_НІК/gerat.git`
2. Створіть та активуйте віртуальне середовище:
   `python -m venv venv`
   `venv\Scripts\activate` (для Windows)
3. Встановіть залежності:
   `pip install -r requirements.txt`
4. Запустіть сервер:
   `python app.py`
5. Відкрийте браузер за адресою `http://127.0.0.1:5000`
